# Cold-Chain-Logistics-Reverse-Logistics-Optimization
Project Overview:
This project is designed to optimize cold chain management and reverse logistics processes in the pharmaceutical industry. By analyzing real-time temperature data during transport, the project identifies cold chain violations and manages the return processes for damaged or spoiled products.
Key Features:
SQL Database: Relational data management across Shipments, ColdChainLogs, and Inventory_Status tables.
Data Analysis: Detection of temperature violations exceeding critical thresholds (8°C).
Visualization: Generation of shipment and route-based temperature performance charts to support decision-making.
Technologies Used:
Python (Pandas, Matplotlib)
SQL (SQLite)
